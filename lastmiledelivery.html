Same day & Last Mile Delivery

How it Works:

  • The user, via the Blink App provides cargo/parcel details to be shipped.
  • The Blink App then provides a quote across a number of available cargo shipping agents from where the Cargo/Parcel would be delivered for onward processing.
  • The App will enable the user to establish the most effective price combination of ‘Cargo Agent to Cargo Airline’ based on dimensions, weight, and destination.
  • Blink 'Last Mile Delivery' driver will pick up the package from the origin (shipper destination) and have the cargo delivered to an identified Cargo Agent and later from the Cargo Agent delivered to the given Cargo Airline.
  • At destination point, the Blink Captain will pick up the Cargo/Package from the Cargo destination Agent for delivery to a given destination as requested via the App.
